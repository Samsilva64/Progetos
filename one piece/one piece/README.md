# One Piece Login

Tela de login com identidade visual inspirada em One Piece, usando imagens personalizadas e formulario estilizado.

## Funcionalidades

- Campo de usuario.
- Campo de senha.
- Opcao `Remember me`.
- Link de recuperacao de senha.
- Botao de login com imagem.
- Link para registro.
- Layout com imagem de fundo, icones e efeito de transparencia.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Como executar

1. Abra a pasta `one piece/one piece`.
2. Abra o arquivo `index.html` no navegador.
3. Interaja com o formulario.

## Estrutura

```text
one piece/
  index.html
  style.css
  script.js
  img/
    acesso.png
    chapeu.png
    email.png
    fundo.jpg
    login.png
    logo1.png
    logo2.png
    nome.png
    titulo.png
```

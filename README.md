# Projetos Front-end

Este repositorio reune tres projetos simples desenvolvidos com HTML, CSS e JavaScript puro.

## Projetos

- [Restaurante La Rosa](./la%20sosa/la%20sosa/README.md): cardapio digital com pedidos via WhatsApp e painel local para gerir pratos.
- [Login](./login/login/README.md): formulario de login/cadastro simples com validacao basica do navegador.
- [One Piece Login](./one%20piece/one%20piece/README.md): tela de login com tema visual inspirado em One Piece.

## Como usar

1. Abra a pasta do projeto desejado.
2. Abra o arquivo `index.html` no navegador.
3. Edite `style.css` para alterar o visual e `script.js` para ajustar comportamentos.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Observacao

Os projetos rodam diretamente no navegador e nao precisam de instalacao de dependencias.

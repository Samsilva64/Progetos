# Login

Formulario simples de login/cadastro feito com HTML, CSS e JavaScript.

## Funcionalidades

- Campos para nome, sobrenome e email.
- Escolha de linguagem favorita com botoes de radio.
- Validacao basica usando atributos `required`.
- Captura do envio do formulario com JavaScript.
- Exibicao do primeiro nome no console do navegador.
- Fundo com imagem e estilo visual com efeito de transparencia.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Como executar

1. Abra a pasta `login/login`.
2. Abra o arquivo `index.html` no navegador.
3. Preencha o formulario e clique em `Login`.
4. Abra o console do navegador para ver o valor capturado pelo JavaScript.

## Estrutura

```text
login/
  fundo.jpg
  index.html
  style.css
  script.js
```

## Observacao

Este projeto ainda nao possui autenticacao real. Ele serve como base visual e estrutural para um formulario.
